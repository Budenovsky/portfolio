# 🍳 RecipeGA

**RecipeGA** — это веб- и мобильное приложение для любителей готовить.  
Здесь пользователи создают, хранят и делятся рецептами с фотографиями, полным составом блюд и точной пищевой ценностью на основе КБЖУ (калории, белки, жиры, углеводы).

---

## ✨ Возможности пользователей

### 1. 🔐 Регистрация в системе  
Быстрое создание личного профиля для доступа ко всем функциям платформы.

---

### 2. 🥑 Создание продуктов с указанием КБЖУ  
Пользователи могут добавлять собственные продукты с указанием:
- Калорийности  
- Белков  
- Жиров  
- Углеводов  

---

### 3. 📝 Добавление рецептов  
Каждый рецепт может включать:

- 🍽️ **Название блюда**  
- 📖 **Описание**  
- 🧾 **Список ингредиентов**  
- 👨‍🍳 **Пошаговая инструкция приготовления**  
- 🖼️ **Основная и дополнительные фотографии**

---

### 🔄 Гибкость в выборе продуктов  
Для создания рецептов пользователи могут:
- ✅ Создавать **собственные продукты**  
- ✅ Использовать **готовые продукты** из общего справочника  
- ✅ Брать продукты, **добавленные другими пользователями**

---

> ✨ Просто делитесь кулинарными идеями, вдохновляйте других и следите за питательной ценностью каждого блюда!
